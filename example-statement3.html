Accessibility Statement for Citylights Website

Citylights Inc. is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone, and applying the relevant accessibility standards.

Measures taken

We took the following measures:

  • Include accessibility throughout our internal policies.
  • Assign clear accessibility targets and responsibilities.

Conformance status

The Web Content Accessibility Guidelines (WCAG) have three conformance levels (A, AA and AAA). These guidelines provide criteria to make web content more accessible for everyone, including for people with disabilities. https://www.w3.org/WAI/demos/bad/after/home.html is Partially conformant with the WCAG 2.1 AA standard. Partially conformant means that Some parts of the content do not fully conform to the accessibility standard (you can indicate these parts in later sections of this form).

Additions to the requirements

Although our goal is level AA conformance, we have also applied some level AAA success criteria. For example, images of text are only used for decorative purposes. When an authenticated session expires, the user can continue the activity without loss of data after re-authenticating. We added sign language in all our videos.

Technical specifications

The technologies this website uses may affect the usability of the website. This also depends on the combination of assistive technologies, browsers and plugins that are used. We used the following technologies to measure the accessibility of this website:

  • HTML
  • WAI-ARIA
  • CSS
  • Javascript
  • SMIL

These technologies are relied upon for conformance.

Compatibility with browsers and assistive technology

The website is designed to be compatible with the following assistive technologies:

  • Firefox v62 with NVDA Assistive Technology v2018.3.2
  • Safari v12 with VoiceOver Assistive Technology

The website is not compatible with:

  • Firefox v62 with VoiceOver Assistive Technology

Limitations and alternatives

Depending on the combinations of browsers, plugins and assistive technology, you may still experience issues with the use of this website/mobile application. In some cases, we are working on solutions for know issues. Please try the solutions provided below or if you experience that this list is not complete, contact us.

  • Homepage: Multiple images without a textual alternative because Editorial mistakes. Repair in 2 days. Contact us.
